The Smart Card Shell is provided as zipped Windows Installer (MSI) file or as a zipped archive with the necessary files and directories. Use the former if you are on a Windows machine and have full access to install software. Use the later if you want to manually copy the files onto your disk or plan to use the Smart Card Shell on Linux.
