HttpResponse - Reference Documentation

Class implementing the response passed to a scripting servlet.

Instances of this class wrap the HttpServletResponse from the Servlet API and allow scripts easy access to properties of an outgoing HTTP response.

Instances of this class can not be created directly, but are passed as argument to the handleRequest() method.
